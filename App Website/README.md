# Guardian App — Blank Website

This repository contains a minimal placeholder website for the Guardian App.

Files included:
- `index.html` — homepage that links the CSS and JS
- `assets/css/style.css` — styles
- `assets/js/script.js` — small client script
- `assets/img/` — place images here (app icon, badges)

Using the provided app icon

Place the app icon image into the `assets/img/` folder with the filename:

```
assets/img/app-icon.png
```

The header will use `assets/img/app-icon.png` as the site logo. Recommended size: 1024×1024 (or at least 512×512). The file can be PNG or SVG.

Quick start

Open the site locally by either:

1) Opening `index.html` directly in your browser (double-click it).

2) Running a simple HTTP server (recommended) from the project directory:

```bash

# Python 3
python3 -m http.server 8000

# then open http://localhost:8000 in your browser
```
