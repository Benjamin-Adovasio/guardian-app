# Guardian App — Blank Website

This repository contains a minimal placeholder website for the Guardian App.

Files included:
- `index.html` — minimal homepage that links the CSS and JS
- `style.css` — basic responsive styling
- `script.js` — tiny DOM-ready demo script

Quick start

Open the site locally by either:

1) Opening `index.html` directly in your browser (double-click it).

2) Running a simple HTTP server (recommended) from the project directory:

```bash
# Python 3
python3 -m http.server 8000

# then open http://localhost:8000 in your browser
```

Next steps

- Replace the placeholder content in `index.html` with your app UI.
- Add assets and build tooling as needed.
